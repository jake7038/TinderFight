# TinderFight 🥊🔥

TinderFight is a fun and experimental web application inspired by the swipe-based interaction of Tinder and the underground atmosphere of Fight Club. The project was developed as part of the **Web Projects course** during my undergraduate studies at **CEFET-RJ (Federal Center for Technological Education of Rio de Janeiro)**.

The goal of the project was to practice modern web development concepts while building an interactive and engaging user interface.

---

## 📌 About the Project

TinderFight is a swipe-style application where users can browse through fighters and decide whether they would match for a challenge or training with them or skip them — similar to the swipe mechanics popularized by Tinder.

The project focuses on:

- Interactive UI
- Card-based swipe mechanics
- Frontend web development

This project was created purely for **educational purposes**.

---

## 🎯 Features

- 🥊 Swipe-based interaction
- 👤 Fighter profile cards
- ❤️ Accept or ❌ skip mechanics
- 🎨 Fight Club–inspired theme
- 📱 Responsive interface
